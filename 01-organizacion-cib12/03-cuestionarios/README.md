# Cuestionarios semanales

Esta nota depende de la lectura de cada semana segun el silabo y la exposicion del grupo correspondiente.

## Material base

- Silabo: `curso/00 src/silabo.pdf`
- Libro principal de lectura actual: `curso/00 src/libros/huawei-2022.pdf`

## Flujo simple

1. Identificar el capitulo de la semana.
2. Preparar resumen de conceptos, procesos y terminos.
3. Anotar posibles preguntas de control.
4. Guardar un resumen breve por semana.
