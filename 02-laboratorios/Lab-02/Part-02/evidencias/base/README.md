# Evidencia Base

Esta carpeta guarda la evidencia principal y resumida de la reproduccion base del experimento.

## Archivos esperados

- `comandos-ejecutados.txt`
- `metricas-reproduccion.csv`

## Uso

Sirve como respaldo rapido para:

- mostrar los comandos reales usados,
- resumir metricas de la reproduccion,
- y sostener la trazabilidad del pipeline sin abrir toda la carpeta `resultados`.
