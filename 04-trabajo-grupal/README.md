# Trabajo Grupal y Proyecto Final

Esta carpeta es ahora la ubicacion canonica del trabajo de investigacion grupal del curso.

Se unifican aqui:

- el avance del tema de investigacion;
- la construccion progresiva del articulo cientifico;
- la presentacion final;
- y el material del proyecto final.

## Tema del grupo

Tema base: modelo de aprendizaje profundo con mecanismos de atencion para mejorar la calidad del servicio de guia virtual con aplicaciones moviles e IoT con senales Wi-Fi/BLE en museos y espacios culturales.

## Estructura

- `docs/`: avances, revision, lineamientos, matriz y metodologia
- `datos/`: insumos del proyecto grupal
- `src/`: codigo fuente o scripts
- `notebooks/`: exploracion y experimentacion
- `resultados/`: tablas, metricas, figuras y salidas
- `informe/`: articulo cientifico, borradores y version final
- `presentacion/`: diapositivas y material de sustentacion

## Logica de trabajo

1. Definir y ajustar el tema.
2. Consolidar antecedentes.
3. Completar matriz de consistencia.
4. Definir metodologia y plan experimental.
5. Recolectar datos y ejecutar experimentos.
6. Redactar resultados, discusion y conclusiones.
7. Preparar el articulo final, la traduccion y la exposicion.
