# Prompt para NotebookLM - Semana 06

## Objetivo de esta semana

Desarrollar marco teorico y revisar guia para autores de una revista objetivo.

## Archivos que debes cargar en NotebookLM

- papers y antecedentes ya seleccionados
- `04-trabajo-grupal/informe/articulo-base.md`
- si existe, la matriz validada de la semana 05
- guia para autores de la revista que el grupo piense usar
- si existen, entregables previos en `entrega/`

## Prompt sugerido

Actua como asistente academico para redaccion cientifica. Quiero preparar el entregable de la semana 06: marco teorico y guia para autores.

Tema del grupo:
"Modelo de aprendizaje profundo con mecanismos de atencion para mejorar la calidad del servicio de guia virtual con aplicaciones moviles e IoT con senales Wi-Fi/BLE en museos y espacios culturales, 2026".

Con el material cargado, necesito que:

1. Organices el marco teorico en subsecciones coherentes.
2. Identifiques conceptos clave que deben definirse.
3. Propongas una narrativa academica que conecte teoria, antecedentes y problema.
4. Resumas la guia para autores de la revista elegida en terminos utiles para escribir el paper.

Entrega la salida en este formato:

- bloque 1: esquema del marco teorico;
- bloque 2: parrafos base por subseccion;
- bloque 3: checklist de la guia para autores;
- bloque 4: esquema para una presentacion semanal;
- bloque 5: lista de elementos que deben respetarse luego en el articulo final.

## Uso posterior

Con esta informacion se puede redactar el marco teorico formal y convertir el resumen a una presentacion semanal en `.tex`.
