# Prompt para NotebookLM - Semana 14

## Objetivo de esta semana

Preparar traduccion, preprint y revision final.

## Archivos que debes cargar en NotebookLM

- articulo final completo
- version en espanol
- lineamientos de publicacion o preprint
- evidencias o datos del repositorio donde se subira
- entregables previos dentro de `entrega/`

## Prompt sugerido

Actua como revisor final de un articulo cientifico en etapa de cierre. Quiero preparar el entregable de la semana 14: traduccion del articulo, preprint y revision final.

Tema del grupo:
"Modelo de aprendizaje profundo con mecanismos de atencion para mejorar la calidad del servicio de guia virtual con aplicaciones moviles e IoT con senales Wi-Fi/BLE en museos y espacios culturales, 2026".

Usa el material cargado para apoyar el cierre del trabajo.

Genera:

1. checklist de revision final del articulo;
2. observaciones para asegurar consistencia entre la version en espanol y la version traducida;
3. recomendaciones de estilo academico para la version final;
4. lista de metadatos y evidencias necesarias para el preprint;
5. resumen de cierre para una presentacion final corta.

Ademas:

- marca posibles errores de terminologia tecnica al traducir;
- verifica que el resumen, titulos y palabras clave sean coherentes;
- sugiere evidencias que conviene guardar para sustentar el DOI o preprint.

## Uso posterior

Con esta salida se puede cerrar el paquete final del curso y, si se necesita, crear una presentacion `.tex` de revision final.
