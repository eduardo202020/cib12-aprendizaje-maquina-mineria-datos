## Prompt sugerido

Actua como asistente academico para un trabajo grupal universitario en Machine Learning y Big Data. Quiero construir el entregable de la semana 02: sistematizacion del tema de investigacion.

Tema del grupo:
"Modelo de aprendizaje profundo con mecanismos de atencion para mejorar la calidad del servicio de guia virtual con aplicaciones moviles e IoT con senales Wi-Fi/BLE en museos y espacios culturales, 2026".

Usa como contexto los antecedentes cargados y el avance actual del tema. Necesito que revises si la sistematizacion esta bien planteada y me propongas una version mejorada, clara y coherente.

Genera lo siguiente:

1. Linea de investigacion.
2. Tema general.
3. Tres temas particulares.
4. Tres temas especificos.
5. Problema de estudio.
6. Titulo preliminar.
7. Titulo tentativo.
8. Variable independiente.
9. Variable dependiente.
10. Objeto de estudio.
11. Alcance espacial.
12. Alcance temporal.

Ademas:

- explica brevemente por que cada elemento es coherente con el tema;
- detecta posibles inconsistencias o ambiguedades;
- sugiere mejoras concretas de redaccion;
- prepara un resumen corto para una presentacion semanal de 3 a 5 diapositivas.

Quiero la salida en este formato:

- bloque 1: version academica final del contenido;
- bloque 2: observaciones y mejoras;
- bloque 3: estructura para presentacion semanal con titulo de cada diapositiva y bullets.
