# Prompt para NotebookLM - Semana 09

## Objetivo de esta semana

Planificar o documentar la recoleccion de datos.

## Archivos que debes cargar en NotebookLM

- metodologia desarrollada hasta la semana 07
- definicion del dataset, si ya existe
- fichas, tablas o notas de capturas de datos
- si existen, entregables previos en `entrega/`

## Prompt sugerido

Actua como asesor de investigacion experimental. Quiero preparar el entregable de la semana 09: recoleccion de datos.

Tema del grupo:
"Modelo de aprendizaje profundo con mecanismos de atencion para mejorar la calidad del servicio de guia virtual con aplicaciones moviles e IoT con senales Wi-Fi/BLE en museos y espacios culturales, 2026".

Con base en el material cargado, necesito que organices la recoleccion de datos de manera clara y defendible academicamente.

Genera:

1. descripcion del origen de los datos;
2. procedimiento de recoleccion o seleccion del dataset;
3. criterios de inclusion, exclusion y limpieza;
4. estructura del conjunto de datos;
5. posibles sesgos o limitaciones de la muestra;
6. propuesta de tabla para documentar el dataset.

Ademas:

- si los datos aun no estan completos, plantea el plan de recoleccion faltante;
- resume que se puede mostrar en la presentacion semanal;
- incluye una lista de evidencias utiles que deberian guardarse.

## Uso posterior

Con esta salida se puede escribir el apartado de datos y preparar una presentacion semanal en `.tex`.
