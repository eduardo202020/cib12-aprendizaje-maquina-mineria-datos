# Prompt para NotebookLM - Semana 10

## Objetivo de esta semana

Definir o explicar los experimentos.

## Archivos que debes cargar en NotebookLM

- metodologia y datos del proyecto
- notas sobre baselines, modelo propuesto, hiperparametros y metricas
- si existen, scripts, tablas o capturas de entrenamiento
- entregables previos dentro de `entrega/`

## Prompt sugerido

Actua como asesor en machine learning aplicado. Quiero preparar el entregable de la semana 10: ejecucion de experimentos.

Tema del grupo:
"Modelo de aprendizaje profundo con mecanismos de atencion para mejorar la calidad del servicio de guia virtual con aplicaciones moviles e IoT con senales Wi-Fi/BLE en museos y espacios culturales, 2026".

Con el material cargado, necesito que estructures claramente el apartado experimental.

Genera:

1. objetivo experimental;
2. descripcion del baseline;
3. descripcion del modelo propuesto;
4. configuracion experimental;
5. metricas usadas;
6. criterios de comparacion;
7. posibles amenazas a la validez experimental.

Ademas:

- si el experimento aun no esta completo, propone el siguiente paso concreto;
- prepara una version resumida para presentacion semanal;
- sugiere tablas y figuras que convenga mostrar.

## Uso posterior
