# Notebooks

Estos notebooks estan separados por semana para que puedas abrirlos y ejecutarlos directamente.

## Orden recomendado

- `semana-01-datos.ipynb`
- `semana-02-m2-sin-vacios.ipynb`
- `semana-03-m2-con-vacios.ipynb`
- `semana-04-cierre.ipynb`

## Uso

1. Abre el notebook de la semana correspondiente.
2. Ejecuta las celdas de arriba hacia abajo.
3. Si quieres regenerar resultados, descomenta las celdas marcadas como ejecucion opcional.
