# Plantilla de Laboratorio CIB12

Usa esta carpeta como base para cada nuevo laboratorio del curso `CIB12 - Aprendizaje de Maquina y Mineria de Datos`.

## Datos del curso

- Curso: `Aprendizaje de Maquina y Mineria de Datos`
- Codigo: `CIB12`
- Departamento academico: `Telecomunicaciones`
- Condicion: `Electivo`
- Creditos: `3`
- Horas por semana: `HT 2, HL 2`
- Sistema de evaluacion: `M`

## Enfoque del curso

El silabo combina:

- Machine Learning aplicado a telecomunicaciones;
- Big Data por lotes y en tiempo real;
- Deep Learning para imagen, voz y lenguaje;
- investigacion academica aplicada y exposiciones tecnicas.

## Objetivos del curso que puedes reutilizar

- optimizar redes inalambricas y opticas con Machine Learning;
- implementar arquitecturas Big Data para analisis masivo;
- desarrollar soluciones de IA con redes neuronales y Transformers;
- producir investigacion academica aplicada con formato publicable.

## Checklist del laboratorio

- completa `ficha-curso.md`;
- define el problema, objetivo y alcance;
- vincula el trabajo con una unidad del silabo;
- agrega el dataset o su referencia en `datos/`;
- desarrolla el analisis en `notebooks/` o `src/`;
- guarda tablas, graficos y metricas en `resultados/`;
- redacta el informe en `informe/informe.md` o `informe/informe.tex`;
- prepara la exposicion en `presentacion/diapositivas.md`;
- organiza la sustentacion en `presentacion/guion.md`;
- exporta la version final a PDF.

## Recordatorios del silabo

- usa al menos `3 referencias bibliograficas` cuando el trabajo sea monografia, avance o articulo;
- considera `4 diapositivas por participante` para exposiciones;
- apunta a `5 minutos por participante` en la sustentacion;
- si hay demostracion, prepara una demo de `10 minutos`;
- resalta relevancia, sintesis, conclusiones y originalidad.

## Entregables del laboratorio

- Informe final: `../../informes/`
- Presentacion final: `../../presentaciones/`
- Version web de apoyo: `../../docs/presentaciones/`
